# Money-Converter
 ## Objetivo

 Simplificar a Conversão de outras Moedas para o real.

## Moedas
Atualmente Convertemos as seguintes Moedas

 * Dólar Americano
 * Euro
 * Dólar Canadense
 * Libra Esterlina
 * Peso Argentino
 * Iene Japonês
 * Franco Suíço
 * Dólar Australiano
 * Yuan Chinês
 * Novo Shekel Israelense
 * Bitcoin
 * Litecoin
 * Ethereum
 * XRP

 Api Utilizada [AwesomeAPI](https://docs.awesomeapi.com.br/api-de-moedas)
